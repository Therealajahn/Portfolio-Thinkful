Why Do I Code?

I’m really into electronic music, as you can see from my Cosmogramma[make link] quiz. When I looked into software for music making I was turned off by the unintuitive digital audio workstations, and the easy to use hardware methods of making music seemed to be very expensive, lacking flexiblilty, or both. I want to learn to solve problems with code so that I can create software that anyone can access over the web and sell hardware that makes this software more engaging.
